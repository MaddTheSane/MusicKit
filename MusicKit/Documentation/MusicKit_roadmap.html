Where to find information about the MusicKit and SndKit

Nearly all documentation is in SGML DocBook format in the repository, which is then rendered to HTML and uploaded to the homepage, so the homepage is the best URL to visit for nearly all questions regarding the project. It is the README for the project online.

A PDF version of the README/homepage will be bundled in the source release, although at the moment problems with the SGML parser (jade) prevent this.

Documentation for the MusicKit and SndKit are in the process of conversion to DocBook to be available online. The Concepts Book and a set of Tutorials are available and are also accessible via the homepage. Class documentation is currently in RTF and is available in the repository/source releases.

The MusicKit and SndKit are standard sourceforge archives, so learn how to use sourceforge!
